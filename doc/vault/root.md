# Project Oversight

__Project Oversight__ is an umbrella term for several interacting systems which are meant to assist in the operation of a sovereign alliance in EVE Online.

## Development
Languages & Libraries:
- TypeScript + HTML + CSS
- Dgraph database
- Relay implementation of GraphQL
- React + Electron
	- D3
	- OpenChakra
- Snyk
